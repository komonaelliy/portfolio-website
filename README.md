## 🚀 Local Development

```bash
# Clone the repository
git clone https://github.com/komonaelliy/portfolio-website.git

# Navigate to project directory
cd portfolio-website

# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
